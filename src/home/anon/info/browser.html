Browse safely!

You are running a safe browser. All connections are automatically Torified, as is the case with all other network applications in Liberté Linux. The only exception is the unsafe browser, which should be run only for special purposes such as Wi-Fi registration in absence of fully functional Internet connectivity.

Keep in mind that non-https non-onion connections have their payload exposed at Tor exit nodes.
